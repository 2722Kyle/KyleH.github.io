Autonomous Gel Blaster Turret

In this project, we created an autonomous turret using a gel blaster toy gun. The turret is capable of target acquisition and a functional rotational range of 0-270 degrees. Project consists of a 2D LiDAR, PixyCam2, DC motor, and a linear actuator for positioning and sensing capabilities.

Autonomous Vehicle Platform

This project had students create an autonomous vehicular platform to perform in four events as follows: Tug of War, Race, Trade Depot, and Hide n' Seek. To accomplish these tasks, our robot utilized a closed feedback loop for Tug of War, line following for the Race, and color/signature recognition for both Trade Depot and Hide n' Seek.

FPGA Chess/Checkers Implementation

For this project, the group decided to create the game of chess through the use of an FPGA. The game would contain the standard rules associated with chess and would be displayed on a VGA monitor. N-bit ALU in Verilog

For this project, I developed an 8-bit ALU using Vivado and a Zybo-Z7 FPGA board. The ALU itself is capable of addition, subtraction, comparisons between two sets of numbers. The output was done in hexadecimal representation and is scalable to more bits granted for physical checks, more inputs and output ports would be needed.

This project tasked our group with creating an elevator control system. The process took us through design, simulation, and physical testing stages. Different weights are also supported within the elevator car.

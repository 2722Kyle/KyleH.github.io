Additional Projects

Here are some additional projects and lab work that I am proud to share. In progress projects will be marked accordingly.

FPGA Multi-Functional Counter

This project involved creating an 8-bit synchronous counter that had multiple functionalities. The counter itself includes a clocking wizard and dividers in order to display the change in values as well as "flicker" between the two seven segment displays to make them appear to be on at the same time. The counter is capable of loading in a starting value, counting up or counting down, and is enabled through push button inputs on the Zybo-Z7 board.

Java Calculator App

For the final lab/project of my Java course, students were tasked with creating a calculator using Java and the GUI. The base requirements for the calculator was to ensure simple arithmetic functions such as addition, subtraction, multiplication, division, however, I added in more functionalities such as square rooting and such. The calculator worked with some minor bugs and detailed a strong understanding of Java.

2-in-1 Chess and Shogi Game (In progress)

This project could be considered as a passion project. The premise of this project would be done through C++ or Verilog (haven't decided yet) and will be ported over to be a physical game. Users would be able to choose between Chess or Shogi, and have the option to play against a player or an AI of varying difficulty. The project would also display the moves that were made and potentially provide users with feedback on the game/moves. The board itself will feature LEDs to also show legal moves and piece captures. Similar to physical chess boards, but this project aims to streamline the experience for new and veteran players.

Book Data Analyzer w/ Pandas

Simple project to get familiar with data analytics and pandas in Python. Downloaded a .csv file containing data of a variety of books, authors, genres, etc., and exported .csv files with trimmed and reorganized data. This project was meant to aid in developing a foundation using Python, Git, and data analytics. Project can be further improved by further data extraction and conversion to a PowerPoint to automate a slide deck for easy descriptions and analytics.
